# Portfólio — Alícia Maia | QA & Automação

Portfólio pessoal desenvolvido com design **Neobrutalista**. Apresenta projetos, experiências e formas de contato de Alícia Maia, Analista de QA e Automação de Testes.

## 🛠 Stack

- **React 19** + **Vite 8**
- **Tailwind CSS v4** (via `@tailwindcss/vite`)
- **lucide-react** e **react-icons**
- **ESLint 9**

## 🚀 Rodando localmente

```bash
npm install
npm run dev
```

A aplicação estará disponível em `http://localhost:5173`.

## 📦 Scripts

| Comando | Descrição |
|---|---|
| `npm run dev` | Inicia servidor de desenvolvimento com HMR |
| `npm run build` | Gera build de produção em `dist/` |
| `npm run preview` | Serve o build de produção localmente |
| `npm run lint` | Executa ESLint no código-fonte |

## 🎨 Design

O portfólio usa a estética **Neobrutalista**: bordas espessas (`border-4`), sombras offset (`shadow-[6px_6px_0px_#181818]`), tipografia bold/uppercase e uma paleta vintage:

| Nome | Hex |
|---|---|
| Creme | `#F5F1DF` |
| Preto | `#181818` |
| Dourado | `#DBA538` |
| Vermelho Vivo | `#D93635` |
| Pêssego | `#F4CDBC` |
| Marrom | `#68412B` |
